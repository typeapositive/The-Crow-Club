# The Crow Club

The Crow Club é um bot de jogos para Telegram feito em Python. A ideia do projeto é criar experiências divertidas com cara de mesa, mistura de brincadeira e interpretação. No momento, o jogo disponível é o Blackjack.

Só que aqui o Blackjack não é só sobre bater 21. As cartas são amaldiçoadas, e isso muda o clima da partida. Quando alguém estoura, não perde apenas a rodada, também recebe uma maldição. 

Esse projeto nasceu mais pela vontade de criar algo legal do que por qualquer formalidade. É um espaço para testar ideias, praticar Python, explorar mecânicas diferentes e se divertir programando.

## O que o bot é hoje

Hoje, o The Crow Club funciona como um bot de jogos com foco em uma proposta mais narrativa. Por enquanto, ele conta com:

- Blackjack no Telegram
- cartas com maldições
- suporte a uma dinâmica mais voltada para RPG e improviso

## Como funciona a ideia

Os jogadores entram na mesa, compram cartas e tentam chegar o mais perto possível de 21 sem passar. Quando passam, vem a consequência: uma carta amaldiçoada entra em cena e adiciona um efeito que pode ser interpretado no jogo.

A graça está justamente nisso. O bot cuida da estrutura da partida, mas o grupo pode usar os resultados para transformar cada rodada em uma experiência mais interpretativa, engraçada, caótica ou sombria, dependendo da carta que receber.

## Tecnologias

- Python
- `python-telegram-bot`

## Objetivo do projeto

The Crow Club existe para:

- treinar programação em Python
- exercitar criatividade
- testar ideias de jogos
- criar algo divertido de usar com amigos

## Observação

Esse é um projeto experimental, feito com carinho e com vontade de explorar programação de um jeito mais criativo. A proposta é simples: fazer um bot de jogos que seja divertido de construir e divertido de jogar.
